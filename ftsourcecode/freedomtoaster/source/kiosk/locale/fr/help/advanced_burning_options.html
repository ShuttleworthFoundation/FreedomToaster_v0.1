Aide au choix d'une version différente

Sélectionnez simplement la version désirée dans la liste des versions, puis cliquez sur le bouton Sélectionner

Note : le choix d'une version avec ce procédé vous placera en mode expert. Cela signifie que vous aurez la possibilité de graver plusieurs images CD ISO sur un DVD et que vous verrez apparaître une page avec plus d'options pour satisfaire vos souhaits.

Versions disponibles

S'il n'y a qu'une version dans la liste, cela signifie que c'est la seule version disponible sur ce Freedom Toaster

Certaines versions apparaissent hors de l'écran

La liste des versions ne peut afficher que 10 lignes à la fois. S'il y a plsu de versions, vous pouvez remarquer un ascenseur à droite de la liste. Déplacez la cage vers le haut ou vers le bas pour afficher les autres éléments de la liste.

Revenir à l'emplacement précédent

Cliquez simplement le bouton Retour (<) en bas à gauche

Revenir à la page d'accueil

Cliquez simplement le bouton Accueil en haut à gauche de l'écran
