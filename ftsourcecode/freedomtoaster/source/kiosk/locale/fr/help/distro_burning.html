Aide sur les informations des distributions

Cette écran affiche les avantages et les inconvénients de chaque distribution Linux ou de contenu libre.

Gravure de la dernière version

  • Le système propose automatiquement la version la plus récente ou la plus populaire. Cliquez sur le bouton 'Sélectionner' pour utiliser cette version (recommandé pour la plupart des utilisateurs).

Choisir d'autres versions

  • Quelquefois, des versions plus anciennes sont disponibles, ainsi que des versions pour d'autres matériels comme le Mac d'Apple ou d'autres architectures non-Intel
  • Cliquez sur "Autres versions" pour obtenir une liste des autres versions disponibles pour cet élément
  • Choisissez la version alternative dans la liste et cliquez sur 'Sélectionner' pour la placer dans la liste

Annuler la sélection

Si vous ne souhaitez pas ajouter un nouvel élément à la liste de gravure, cliquez sur le bouton 'Accueil' (en haut à gauche de l'écran) pour revenir à la page d'accueil ou sur le bouton 'Précédent' (<) (en bas à gauche) pour revenir à la page précédente.

N'hésitez pas à ajouter à la liste tout élément qui pourrait vous intéresser, car vous aurez toujours l'occasion d'effacer des éléments que vous ne souhaitez plus avant de procéder à la gravure.
